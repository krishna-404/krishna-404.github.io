Chotu - URL Shortner

Chotu is a REST API to get a quick & hassle-free short link generator. Also tracks the number of times the chotu link has been accessed. Attending the chotu link forwards the user them to the Long URL.


REST API allows a user to call an endpoint with the 'Long Url' & an optional 'preferred short code'. The API returns a short link if the preferred code is available or it creates a random short code. Accessing the '/short-code/stats' endpoint gives the statistics about the number of times the url has been used/visited.


Share chotu links instead of Long URLs.
